# My-Tableau-
# 📊 Tableau Data Visualization Project – Retail & Sales

## 📌 Overview
This project was completed as part of a Data Technician bootcamp and focuses on transforming retail and sales data into interactive, visually compelling dashboards using Tableau. The objective was to uncover insights and communicate data effectively through clear and engaging visual storytelling.

---

## 🛠️ Skills & Techniques Demonstrated

### 📊 Dashboard Development
- Designed and built **interactive dashboards** to present key sales metrics  
- Structured layouts for clarity, usability, and user engagement  
- Enabled dynamic exploration of data through user interaction  

---

### 🎛️ Interactivity & Controls
- Implemented **filters and slicers** to allow users to drill down into specific segments  
- Created **interactive elements** to explore data by region, product category, and time period  

---

### 🧮 Calculated Fields
- Developed **calculated fields** to derive new metrics and KPIs  
- Applied logic to enhance analysis and support deeper insights  

---

### 📈 Data Visualisation
Utilized a variety of chart types to represent data effectively:
- **Bar charts** – Comparing product and category performance  
- **Line charts** – Visualizing sales trends over time  
- **Pie charts** – Showing proportional distributions  
- **Maps** – Displaying geographic sales patterns  

---

## 📖 Data Storytelling
- Translated complex datasets into **clear, meaningful narratives**  
- Highlighted key trends, patterns, and business insights  
- Designed visuals to guide users through the data logically  

---

## 📈 Key Outcomes
- Delivered an intuitive dashboard for analyzing retail performance  
- Enabled stakeholders to quickly identify trends and opportunities  
- Improved understanding of sales patterns across time and location  

---

## 🚀 Tools Used
- Tableau (Dashboards, Calculated Fields, Interactive Visuals)

---

## 📂 Project Focus
Retail and sales data analysis, including:
- Revenue and performance tracking  
- Regional and product-level insights  
- Time-based sales trends  

---

## 💡 What I Learned
- How to design effective and interactive dashboards  
- The importance of user-focused data visualization  
- Techniques for enhancing insights using calculated fields  
- Best practices for storytelling with data
